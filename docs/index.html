EverMemOSKit

EverMemOSKit is a standalone Swift SDK for integrating EverMemOS memory workflows into iOS and macOS apps.

Built for Swift

Typed models, async/await APIs, Swift Package Manager distribution.

Memory-native

Write, fetch, search, redact, and stream memory-aware chat in one SDK.

Production-minded

Auth abstraction, retries, SSE support, and automated tests.

How Memory Is Used

The SDK exposes EverMemOS memory operations directly to app developers. Apps can store important user facts and interactions, retrieve structured memories later, and use those results to produce memory-aware chat or behavior.

  1. An app writes a memory with memorize.
  2. The app later fetches or searches memory using typed builders.
  3. The app uses that memory to answer, recommend, or continue context for the user.

How This Helps Users

  • Users do not need to repeat important context every session.
  • Apps can recover prior interactions and preferences with better continuity.
  • Developers can build memory-aware experiences without hand-writing the API layer.
